
## Keyboard shortcuts for Jupyter Notebooks


### Command mode vs. Edit mode
Jupyter Notebooks have two different keyboard input modes:   

 - Command mode as indicated by a grey cell border with a blue left margin.  
 
 - Edit mode as indicated by a green cell border   

### Keyboard short cuts in *Command Mode*
 - shift + enter: run current cell and select below
 - ctrl + enter: run current cell
 - option/alt + enter: run current cell and insert below
 - ESC  then 00 then Ctrl + A the Shift +A: Restart the kernel and re-run all cells
 - A: insert a cell above
 - B: insert a cell below
 - C: copy a cell
 - V: paste a cell
 - DD: delete selected cell
 - shift + M: merge selected cells, or current cell with cell below if only one cell selected
 - II:interrupt the kernel
 - 00: restart the kernel
 - Y: change a cell to code mode
 - M: change a cell to markdown mode
 
### Keyboard short cuts in *Edit Mode*
 - ctrl/cmd + click for multi-cursor editing
 - cmd + / toggle comment lines
 - tab code completion or indent
 - shift + tab tooltip
 - ctrl + shift + - split cell
